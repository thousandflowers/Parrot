Why Parrot

🔒

Fully offline by default

Bundled llama-server runs locally. Your text never leaves your Mac. No telemetry, no analytics, no data collection.

⚡

Works in every app

Mail, Slack, VS Code, Terminal, Figma, Notion — anywhere the macOS Accessibility API can reach. No browser extension needed.

🎯

4 correction modes

Grammar (⌘⇧E), Fluency (⌘⇧T), Translate (⌘⇧U), Writing Coach (⌘⇧W). Plus custom prompts with your own hotkeys.

🌍

50+ languages

Auto-detects the text's language and adapts prompts automatically. Not just English — Italian, Spanish, French, German, and more.

🎨

Diff highlight + Explain

See exactly what changed in green. Ask the model to justify each correction inline. Hear the result with TTS.

🏷️

No subscription

Free and open source (MIT). Bring your own API key for cloud models, or use the bundled local LLM — zero cost forever.

Install

Download the app

Get the latest release from GitHub:

https://github.com/thousandflowers/Parrot/releases/latest

Build from source

Requires macOS 14+, Xcode 16+:

git clone https://github.com/thousandflowers/Parrot
cd Parrot
swift build -c release --arch arm64

Homebrew (coming soon)

Once notarized:

brew install --cask parrot
